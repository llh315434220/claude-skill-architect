# Claude Skill Architect

**Build high-quality Claude Skills that strictly adhere to Anthropic's official "Complete Guide to Building Skills for Claude".**

[English Version](README.md) | [中文版](README_CN.md)

This meta-skill transforms Claude into an expert Skill Architect. It doesn't just write prompts; it engineers robust, modular **Skill Packages** designed for reliability, maintainability, and token efficiency.

## 🚀 Features

*   **Official Best Practices:** Built directly from Anthropic's [Complete Guide to Building Skills for Claude](https://resources.anthropic.com/hubfs/The-Complete-Guide-to-Building-Skill-for-Claude.pdf).
*   **Progressive Disclosure Architecture:** Automatically structures skills into:
    *   **Router Layer:** Concise YAML frontmatter for efficient tool selection.
    *   **Controller Layer:** `SKILL.md` body with clear logic flow.
    *   **Knowledge Layer:** `references/` and `scripts/` for heavy lifting, keeping the context window clean.
*   **Chain of Thought (CoT) Integration:** Enforces `<thinking>` blocks in generated skills to improve reasoning and reduce hallucinations.
*   **Structured Input Schema:** Defines clear XML-based input parameters for every skill.

## 📂 Repository Structure

A standard Skill generated by this Architect looks like this:

```text
my-new-skill/
├── SKILL.md          # The brain: Metadata + Logic + Instructions
├── references/       # The library: Static knowledge, templates, API docs
│   ├── template.md
│   └── glossary.md
└── scripts/          # The hands: Python/Bash scripts for deterministic tasks
    └── utils.py
```

## 📦 Installation

1.  Copy the `SKILL.md` file from this repository into your Claude/OpenClaw skills directory:
    ```bash
    mkdir -p /path/to/your/skills/claude-skill-architect
    cp SKILL.md /path/to/your/skills/claude-skill-architect/
    ```
2.  Reload your agent or restart the session to register the new skill.

## 💡 Usage

Once installed, simply ask Claude to build a skill for you. Be specific about the **trigger** (when it runs) and the **outcome**.

### Example Prompts

**Basic:**
> "Create a skill for writing Python docstrings following Google style guide."

**Advanced:**
> "Build a 'Weekly Report Generator' skill. It should take a list of bullet points as input. It needs a `references/template.md` file to store the report format. The skill should strictly check for missing metrics before generating the report."
